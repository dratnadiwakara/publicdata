#SOI Tax Income

This script cleans the IRS SOI Tax Income data from https://data.nber.org/data/soi-tax-stats-individual-income-tax-statistics-zip-code-data-soi.html.<br/>
<br/>
The output files in the zip folder contains zipcode level summaries for each year. The files are .fst files and can be loaded in to R using read_fst function.
